Data

The covariates contained in are collected from Tennessee's Student Teacher Achievement Ratio (STAR) project, this projects purpose was to examine the effects of class size on student performance. In this experiment teachers are randomly assigned to groups of students which should help ensure the errors in the model are independent of student characteristics. In order to test our models performance we will randomly split our data into an 80% training set and a 20% test set. Preliminary Analysis

There are two main ways we could go about estimate θ. The first way we could do this is by estimating the above model with OLS and teacher specific fixed effects. This would allow us to obtain an unbiased estimate for θ but it won't neccesarily be the most precise estimate of θ. The alternative is to use Maximum Likelihood Estimation with the Empirical Bayes Estimator to get an estimate of θ. The benefit of using the MLE over OLS here is that we can get more precise estimates of the teacher effects.

The main benefit of using the Empirical Bayes over OLS with fixed effects is that the EB estimator is more robust to overfitting. When overfitting is a problem for OLS the EB will give us more accurate coefficient estimates and when overfitting is not a problem then the EB will give us the same estimates as the OLS fixed effects.
