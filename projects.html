Projects

Here is a collection of projects I have worked on over the years!

Predicting Home Prices in King County

  • An RMarkdown presentation on home price prediction using a dataset of home characteristics in King County, WA.
  • I fit a variety of popular machine learning algorithms and compare the results to a multiple linear regression as the baseline.
  • By using a more flexible model we are able to reduce the test error by 40% compared to the linear model.

Flowers Image Recognition

  • In this project I use images of flowers from flickr and google images; the species included are daisies, dandelions, roses, sunflowers, and tulips.
  • Pre-Processed images and designed a pipeline to feed image data into a Convolution Neural Network.
  • Able to predict which of the five classes a flower belongs to based upon a single 224x224 pixel image with ~90% accuracy.
  • Created plots of intermediate activations of the network to visualize how the images were decomposed.

Personal Loan Classification

  • In this RMarkdown presentation I analyze a dataset of account and demograpgic information from a large bank.
  • I attempt to predict which clients will accept an offer for a personal loan to help streamline advertising efforts.
  • Provided a brief exploratory analysis.
  • Created ROC curves to assess model performance.

Orange Juice Analysis and Prediction

  • This project uses data on orange juice purchases from a grocery chain.
  • Using a support vector machine, principal components regression along with a few other methods I attempt to predict whether customers purchase Citrus Hill or Minute Maid OJ.
  • I also incorporate a clustering analysis with k-means and hierachical clustering.

Predicting College Applications

  • This project uses a data on college applications from a variety of schools in the U.S.
  • I attempt to predict total college applications using regression Splines and Generalized Additive Models.
  • In addition, I use bootstrapping to improve the estimated standard errors of the model coefficients.
