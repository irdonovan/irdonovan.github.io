Willingness To Pay

Ian Donovan

Introduction

Streaming services like Netflix, Hulu, and Disney are interested in determining how much their customers value their services. In particular, these streaming services would like to know how much their customers are willing to pay for different products. By estimating the willingness to pay for different product attributes managers would be able to answer questions like should Netflix offer more movies this year or more tv shows? How much will profits increase/decrease at Hulu if we start selling our users data? How much revenue could Disney expect from adding/removing commercials on their streaming service?

Data

To get an estimate of the willingness to pay we will use data from a conjoint analysis on streaming products. The respondants of the survey were asked to rate a set of hypothetical streaming services. Some of the characteristics of these hypothetical services were whether the service sells its users personal data, offers mostly TV shows, offers mostly movies, has commercials, has a fast forward feature, has online streaming availability, and the monthly price of the service. Each individual in the sample is asked to make a choice between five streaming services with given attributes. These questions can be used to extract customers preferences which can then be used to estimated willingness to pay.

Estimation

First, we will estimate a logit model to determine the probability that a respondant chooses a particular service from one of five options based on the price of the service, whether it sells its users personal data, and the other covariates. Next, we can estimate the willingness to pay by dividing the estimated coefficients by the coefficient on the price variable.

Mixed Logit

Now suppose we believe that different customers have different levels of willingness to pay. The simple random utility model would no longer give us the true relationship between the streaming plans and enrollment. To remedy this issue we can use a mixed logit model and allow some of the coefficient estimates on the independent variables from before to be fixed while others will be random. The random coefficients will be the for the variables that we believe have different levels of willingness to pay. One example of this is the commercials variable, it seems reasonable to think that some people despise commercials and would be willing to pay a lot to avoid them while others would be indifferent to commercials and not be willing to pay money to remove them. Post Estimation: Identification of Consumer Preferences

Using the estimates and we will simulate 3000 random draws for each . Next we can calculate the likelihood of observing these random draws, given the data at hand. This vector of probabilities for each is called . Finally we use these probabilities to calculate a weighted average of for each individual.

Likelihood Functions

%{
function estMLE = estimRUM(Y,X,J)
nx = size(X, 2);
function ll = calclike(b)
v = reshape(X*b, J, []);
expv = exp(v);
pr = bsxfun(@rdivide, expv, sum(expv, 1));
pr = reshape(pr, [], 1);
like = pr(Y==1);
ll = sum(log(like));
ll= -ll;
end
opts = optimoptions('fminunc', 'MaxFunctionEvaluations', 1e6...
,'MaxIterations', 1e6, 'Display', 'none');
[bMLE,~,~,~,~,H] = fminunc(@calclike, zeros([nx 1]), opts);
estMLE = struct();
estMLE.b = bMLE;
estMLE.se = sqrt(diag(H \ eye(size(H))));
end
%}
%{
function estMLE = estimMixedLogit(Y,Xf,Xr,pers,J,R)
n = max(pers);
nxf = size(Xf,2);
nxr = size(Xr,2);
split_parms = [nxf nxr ((nxr^2 - nxr)/2 + nxr)];
function est = vecparm(parms)
parms = mat2cell(parms,split_parms,1);
est = struct();
est.alpha = parms{1};
est.gamma = parms{2};
cholDelta = zeros([nxr nxr]);
cholDelta(tril(ones(nxr))==1) = parms{3};
est.Delta = cholDelta*cholDelta';
end
function ll = calclike(parms)
est = vecparm(parms);
rng(0);
like = zeros([n 1]);
for i = 1:n
I = (pers==i);
beta_i = mvnrnd(est.gamma',est.Delta,R);
v = bsxfun(@plus,Xf(I,:)*est.alpha,Xr(I,:)*beta_i');
v = reshape(v,J,[]);
expv = exp(v);
pr = bsxfun(@rdivide,expv,sum(expv,1));
pr = reshape(pr,[],R);
like_i = pr(Y(I)==1,:);
like(i) = mean(prod(like_i,1));
end
ll = sum(log(like));
ll = -ll;
end
opts = optimset('MaxIter',1e6,'MaxFunEvals',1e6,'Display','none');
[bMLE,~,~,~,~,H] = fminunc(@calclike,.1*rand([sum(split_parms) 1]),opts);
estMLE = vecparm(bMLE);
estMLE.se = sqrt(diag(H\eye(size(H))));
end
